The Chip-8 uses a 16-key layout. The layout is mapped as follows:


Controls:

  • Airplane: [S]=Drop
  • Astro Dodge: [W]=Start // [Q]=Left // [E]=Right
  • Blinky: [A]=Left // [S]=Right // [3]=Up // [E]=Down
  • Brix: [Q]=Left // [E]=Right
  • Cave: [V]=Start // [Q]=Left // [E]=Right // [2]=Up // [S]=Down
  • Landing: [S]=Drop
  • Lunar Lander: [2]=Thrust // [Q]=Left // [E]=Right
  • Missile: [S]=Fire
  • Paddles: [V]=Start // [Q]=Left // [E]=Right
  • Pong: [1]=Up // [Q]=Down
  • Space Invaders: [W]=Start+Fire // [Q]=Left // [E]=Right
  • Tank: [W]=Fire // [Q]=Left // [E]=Right // [S]=Up // [2]=Down
  • Tetris: [W]=Left // [E]=Right // [Q]=Turn // [A]=Drop
  • UFO: [Q]=Fire Left // [W]=Fire Up // [E]=Fire Right
  • Vertical Brix: [A]=Start // [1]=Up // [Q]=Down
  • Wipe Off: [Q]=Left // [E]=Right
  • Worm: [Q]=Left // [E]=Right // [2]=Up // [S]=Down
  • X-Mirror: [Q]=Left // [E]=Right // [2]=Up // [S]=Down

Controls:

  • Airplane: [8]=Drop
  • Astro Dodge: [5]=Start // [4]=Left // [6]=Right
  • Blinky: [7]=Left // [8]=Right // [3]=Up // [6]=Down
  • Brix: [4]=Left // [6]=Right
  • Cave: [F]=Start // [4]=Left // [6]=Right // [2]=Up // [8]=Down
  • Landing: [8]=Drop
  • Lunar Lander: [2]=Thrust // [4]=Left // [6]=Right
  • Missile: [2]=Thrust // [4]=Left // [6]=Right
  • Paddles: [F]=Start // [4]=Left // [6]=Right
  • Pong: [1]=Up // [4]=Down
  • Space Invaders: [5]=Start+Fire // [4]=Left // [6]=Right
  • Tank: [5]=Fire // [4]=Left // [6]=Right // [8]=Up // [2]=Down
  • Tetris: [5]=Left // [6]=Right // [4]=Turn // [7]=Drop
  • UFO: [4]=Fire Left // [5]=Fire Up // [6]=Fire Right
  • Vertical Brix: [7]=Start // [1]=Up // [4]=Down
  • Wipe Off: [4]=Left // [6]=Right
  • Worm: [4]=Left // [6]=Right // [2]=Up // [8]=Down
  • X-Mirror: [4]=Left // [6]=Right // [2]=Up // [8]=Down
