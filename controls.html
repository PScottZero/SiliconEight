The Chip-8 uses a 16-key layout. The layout is mapped as follows:


Controls:

  • Airplane:   [S]=Drop
  • Astro Dodge:   [W]=Start // [Q]=Left // [E]=Right
  • Blinky: [A]=Left // [S]=Right // [3]=Up // [E]=Down
  • Brix: [Q]=Left // [E]=Right
  • Cave: [V]=Start // [Q]=Left // [E]=Right // [2]=Up // [S]=Down
  • Landing: [S]=Drop
  • Lunar Lander: [2]=Thrust // [Q]=Left // [E]=Right
  • Paddles: [V]=Start // [Q]=Left // [E]=Right
  • Pong: [1]=Up // [Q]=Down
  • Space Invaders: [W]=Start+Fire // [Q]=Left // [E]=Right
  • Tank: [W]=Fire // [Q]=Left // [E]=Right // [S]=Up // [2]=Down
  • Tetris: [W]=Left // [E]=Right // [Q]=Turn // [A]=Drop
  • UFO: [Q]=Fire Left // [W]=Fire Up // [E]=Fire Right
  • Vertical Brix: [1]=Up // [Q]=Down
  • Wipe Off: [Q]=Left // [E]=Right
  • Worm: [Q]=Left // [E]=Right // [2]=Up // [S]=Down
  • X-Mirror: [Q]=Left // [E]=Right // [2]=Up // [S]=Down
